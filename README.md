# Tobogi-studio
Follow the instructions to interact with the project.

## Clone the project
```
git clone git@github.com:luctst/tobogi-studio.git
```

## Start strapi for managing your api
Run the following command line:
```
cd api/
npm install
npm develop
```

## Start the front-end
Run the following command line:
```
cd client/
npm install
npm start
```
